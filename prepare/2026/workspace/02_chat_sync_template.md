# 新对话快速同步模板

每次开启新对话时，尽量先贴这份精简信息，帮助 AI 快速恢复上下文。

你可以直接复制下面模板：

```md
我正在准备 2026 年的面试，请按长期面试教练模式协助我。

## 1. 目标信息
- 目标岗位：
- 目标级别：
- 目标公司：
- 预计面试时间：

## 2. 当前状态
- 技术主线当前阶段：
- 算法专项当前阶段：
- 离最近一场面试还有：
- 最近在复习：
- 最近卡住的问题：

## 3. 重点背景
- 核心技术栈：
- 最重要项目：
- 当前短板：

## 4. 上次结论
- 技术主线上次完成了：
- 算法专项上次完成了：
- 上次遗留任务：

## 5. 这次希望你帮我做什么
- 例如：制定本周计划 / 复习 JVM / 打磨项目介绍 / 做一场模拟面试 / 复盘回答
```

如果你时间很赶，可以只发超短版：

```md

面试时间：
技术主线进度：
算法专项进度：
当前短板：

```
