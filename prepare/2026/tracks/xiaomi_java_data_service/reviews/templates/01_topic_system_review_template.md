## 3. 核心重点展开

每个重点都至少写清下面四件事：

### 3.1 重点 A

- 是什么
- 为什么有它 / 它解决什么问题
- 怎么理解它的机制或流程
- 面试时怎么讲才不空

### 3.2 重点 B

- 是什么
- 为什么有它 / 它解决什么问题
- 怎么理解它的机制或流程
- 面试时怎么讲才不空

## 6. 自测清单

- 你能不能解释这块到底在解决什么问题
- 你能不能把最核心机制讲成一条因果链
- 你能不能说明这块最容易混淆的边界
- 你能不能用 1-2 分钟把它讲顺
