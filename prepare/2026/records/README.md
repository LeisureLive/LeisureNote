# 2026 年度记录说明

这个目录放 2026 这一轮面试准备的执行记录。

当前文件：

- `00_weekly_progress.md`
  - 周维度进展、薄弱点和下周优先级
- `01_interview_log.md`
  - 模拟面试 / 真实面试记录
- `02_daily_execution_checklist.md`
  - 每日执行清单与明日衔接

和其他层的区别：

- 与 `prepare/common/templates/` 相比：
  - 这里是“带时间上下文的实际记录”
- 与 `prepare/2026/tracks/` 相比：
  - 这里更偏执行过程记录，不承载某个岗位 track 的专题正文

推荐使用顺序：

1. 先看年度画像：
   - `prepare/2026/profile/00_interview_master.md`
2. 再按日常推进使用：
   - 周级：`00_weekly_progress.md`
   - 每日：`02_daily_execution_checklist.md`
   - 面试后：`01_interview_log.md`
