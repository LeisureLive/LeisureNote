# 2026 面试准备工作台

这套模板用于解决和 AI 长期协作时“上下文容易丢失”的问题。

核心原则：

- 用固定文件保存长期稳定信息，而不是依赖单次对话记忆
- 每周更新阶段进展，每次对话前快速同步
- 每次结束沉淀结论和下一步，保证准备过程连续

建议使用顺序：

1. 先填写 `prepare/2026/00_interview_master.md`
2. 每周更新 `prepare/2026/01_weekly_progress.md`
3. 每次开启新对话前，基于 `prepare/2026/02_chat_sync_template.md` 贴给 AI
4. 每次模拟面试或真实面试后，记录到 `prepare/2026/03_interview_log.md`
5. 按 `prepare/2026/04_xiaomi_java_data_service_1month_plan.md` 执行 1 个月冲刺计划
6. 每天用 `prepare/2026/05_daily_execution_checklist.md` 做最小复盘
7. 每次开启新对话或复盘时，直接复制 `prepare/2026/06_session_sync_templates.md` 中的模板

建议维护节奏：

- 主档案：信息变化时更新
- 周进展：每周更新 1 次
- 对话同步：每次新对话使用
- 面试记录：每场模拟/真实面试后更新

你也可以后续再按岗位拆分子目录，例如：

- `prepare/2026/java-backend/`
- `prepare/2026/bigdata/`
- `prepare/2026/interview-company-a/`
