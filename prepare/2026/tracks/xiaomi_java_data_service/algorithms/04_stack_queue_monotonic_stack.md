# 算法专题 04：栈 / 队列 / 单调栈

适用目标：

- 理解 `LIFO` 和 `FIFO` 两种结构分别适合什么问题
- 建立“括号匹配、最近更大 / 更小元素、区间展开”这几类高频模式
- 避免把单调栈题误写成暴力双重循环

## 1. 栈和队列的核心特点

- 栈：后进先出，最适合处理“最近一次状态”
- 队列：先进先出，最适合处理“按顺序流入流出”的场景
- 单调栈：栈内元素保持单调，适合快速找到“左边 / 右边第一个更大或更小的元素”
- 很多题表面看是数组题，本质上是“历史状态如何被压缩保存”

## 2. 什么题应该优先想到栈 / 队列

如果题目有下面这些信号，通常优先考虑：

- “括号是否匹配”
- “表达式求值”
- “最近一个更大 / 更小元素”
- “柱状图面积”
- “滑动窗口最大值”
- “按到达顺序处理任务”

## 3. 常考模式

### 3.1 普通栈：匹配、回退、延迟处理

适用场景：

- 括号匹配
- 逆波兰表达式
- 路径化简

代表题：

- `20. 有效的括号`
- `150. 逆波兰表达式求值`
- `71. 简化路径`

核心理解：

- 栈保存的是“后面还要回来用”的上下文
- 不是所有入栈出栈都叫栈题，关键是当前处理依赖最近未消化状态

### 3.2 队列：顺序处理与窗口推进

适用场景：

- 按顺序消费任务
- 层序遍历
- 滑动窗口中的候选下标维护

代表题：

- `232. 用栈实现队列`
- `225. 用队列实现栈`
- `239. 滑动窗口最大值`

### 3.3 单调栈：找最近更大 / 更小元素

适用场景：

- 每个位置都要找左边 / 右边第一个更大值
- 需要快速确定某个元素向左右能扩到哪里

代表题：

- `739. 每日温度`
- `496. 下一个更大元素 I`
- `503. 下一个更大元素 II`

核心理解：

- 栈里放的通常不是“答案”，而是“还没确定答案的候选元素或下标”
- 一旦当前元素让栈顶失去继续保留的资格，就出栈并结算答案

### 3.4 单调栈求区间边界

适用场景：

- 需要求某个元素作为最小值 / 最大值时能影响的区间
- 需要把二维或区间问题压成“边界扩展”问题

代表题：

- `84. 柱状图中最大的矩形`
- `42. 接雨水`
- `85. 最大矩形`

## 4. 栈 / 队列题的高频坑点

- 栈里该存“值”还是“下标”没想清楚
- 单调栈中“严格大于”和“大于等于”条件写混
- 忘记处理遍历结束后栈里剩余元素
- 队列题只会用普通队列，却意识不到需要单调队列
- 层序遍历时把当前层和下一层节点混在一起

## 5. 栈 / 队列题的面试表达模板

```text
这题的关键不是暴力向两边找，而是把还没确定答案的元素先压进栈里。
我会维护一个单调递减（或递增）的栈，栈里存下标而不是值，
因为后面计算答案时通常还需要位置信息。
当当前元素打破栈内单调性时，就说明它可以帮助栈顶元素结算答案。
这样每个元素最多进栈出栈一次，整体复杂度可以做到 O(n)。
```

## 6. 栈 / 队列 / 单调栈专题建议掌握的代表题

### 入门必做

- `20. 有效的括号`
- `232. 用栈实现队列`
- `225. 用队列实现栈`
- `739. 每日温度`

### 面试高频

- `84. 柱状图中最大的矩形`
- `239. 滑动窗口最大值`
- `496. 下一个更大元素 I`

### 进阶

- `42. 接雨水`
- `85. 最大矩形`
- `155. 最小栈`

## 7. 你复习这块时最低要做到什么程度

- 能区分什么时候该用普通栈，什么时候该用单调栈
- 能说清单调栈里为什么常存下标
- 能说清“每个元素最多进栈出栈一次，所以是 O(n)”
- 能独立写出 `20` 和 `739` 这两类最基础模板

## 8. 本专题练习规划

### 第一阶段：模板入门

1. `20. 有效的括号`
   - 训练重点：
     - 普通栈保存未消化状态
     - 入栈 / 出栈匹配规则
2. `739. 每日温度`
   - 训练重点：
     - 单调栈基础模板
     - 栈里为什么存下标

### 第二阶段：高频主战

1. `496. 下一个更大元素 I`
   - 训练重点：
     - 最近更大元素模板
     - 单调栈如何配合哈希表输出答案
2. `239. 滑动窗口最大值`
   - 训练重点：
     - 单调队列
     - 区分“普通队列”和“维护最值的队列”
3. `84. 柱状图中最大的矩形`
   - 训练重点：
     - 单调栈求左右边界
     - 训练“弹栈时结算答案”的思维

### 第三阶段：变体 / 验收

1. `42. 接雨水`
   - 训练重点：
     - 同一道题可从双指针 / 单调栈角度理解
     - 训练跨专题迁移能力
2. `85. 最大矩形`
   - 训练重点：
     - 把二维问题转成多次 `84`
     - 检查你是否真的理解了单调栈边界模型

## 9. 专题验收标准

- 能区分：
  - 普通栈
  - 普通队列
  - 单调栈
  - 单调队列
- 能讲清 `739` 和 `84` 的单调栈分别在求什么
- 能解释为什么单调栈通常是 `O(n)`
- 遇到“最近更大 / 更小元素”或“区间边界”题时，能主动想到单调栈
